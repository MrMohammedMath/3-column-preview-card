### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/3-column-preview-card.git

  cd 3-column-preview-card

  npm install
```

- for running dev server

```bash
npm run start
```

- for build & preview

```bash
npm run build && npm run preview
```
